The Seabase project is made possible by a grant from the Prince Albert II of Monaco Fondation.
Seasbase is a website using gage, a configuration framework for producing "G"enomic "P"ages. It wraps [jBrowse 1](https://jbrowse.org/jbrowse1.html) and SequenceServer in website generated by a Jekyl wrapper. A build script faciliates data and metadata validation and staging.
